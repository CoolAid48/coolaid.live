### CoolAid's Mod List for Hardcore Minecraft

The following list contains every single mod that CoolAid uses to play Minecraft aesthetically pleasing and buttery smooth! Most mods can be found on **Modrinth**, as well as **Curseforge**. Any other questions, please please PLEASE refer to Google and Youtube, not Twitch chat, Thank you! 

*Please note that all of these mods use **FabricMC**, check out [fabricmc.net](https://fabricmc.net/) for any information on what FabricMC is and/or how to install it. Also note that the **Fabric API** (listed) is required for almost any Fabric mod to work!* **All packs are, of course, 1.21.7 compatible!**
