CoolAid's Mod List for Hardcore Minecraft ​

The following list contains every single mod that CoolAid uses to play Minecraft aesthetically pleasing and buttery smooth! Most mods can be found on Modrinth, as well as Curseforge. Any other questions, please please PLEASE refer to Google and Youtube, not Twitch chat, Thank you!

Please note that all of these mods use FabricMC, check out fabricmc.net for any information on what FabricMC is and/or how to install it. Also note that the Fabric API (listed) is required for almost any Fabric mod to work! All packs are, of course, 1.21.7 compatible!

Configuration/Library/Dependency Mods: ​

  • Fabric API – https://modrinth.com/mod/fabric-api (NECESSARY FOR ALL OTHER MODS IN THIS LIST TO WORK)
  • Cloth Config – https://modrinth.com/mod/cloth-config (DEPENDENCY MOD required for those in YELLOW)
  • Searchables – https://modrinth.com/mod/searchables (DEPENDENCY MOD required for those in BROWN)
  • MaLiLib – https://modrinth.com/mod/malilib (DEPENDENCY MOD required for those in RED)
  • Fabric Language Kotlin – https://modrinth.com/mod/fabric-language-kotlin (DEPENDENCY MOD FOR ZOOMIFY)
  • YetAnotherConfigLib – https://modrinth.com/mod/yacl (DEPENDENCY MOD FOR ZOOMIFY)
  • Mod Menu – https://modrinth.com/mod/modmenu
  • Cubes Without Borders – https://modrinth.com/mod/cubes-with-borders
  • Nuit (FabricSkyBoxes) – https://modrinth.com/mod/fabricskyboxes
